Exemplos de tags para um documento HTML

Para criar um texto em HTML, você deve colocar o texto preferencialmente dentro de uma tag de parágrafo, para manter o texto de forma padronizada e formatada no documento. Por isso, para esse documento com exemplos, os momentos em que tiver textos e não forem necessários algo que o destaque, será usado a tag de parágrafo.
